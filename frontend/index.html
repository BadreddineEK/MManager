💾 Sauvegarde & Restauration

Exportez l'intégralité des données de votre mosquée en un fichier ZIP contenant 8 fichiers CSV (familles, enfants, paiements école, adhérents, cotisations, trésorerie…). Vous pouvez ensuite restaurer ce fichier pour réimporter les données manquantes (fusion — aucun doublon n'est créé).

📤 Restaurer depuis un fichier ZIP

Sélectionnez un fichier .zip généré par cette application. Les enregistrements existants ne seront pas écrasés.
