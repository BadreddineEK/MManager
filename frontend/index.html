🚀 Onboarding

Administration — Import initial des données de votre mosquée

ℹ️ À quoi sert cette section ?

Cette section permet d'importer les données existantes de votre mosquée lors de la première mise en service.
Téléchargez le template CSV correspondant, remplissez-le dans Excel ou Google Sheets, puis importez-le.
Chaque import propose un mode simulation (dry run) pour vérifier les données avant de les enregistrer définitivement.
L'import bancaire mensuel (relevés de compte) se fait depuis la section Trésorerie → 🏦 Import bancaire.

Type de données :
🤝 Import adhérents ⬇ Template CSV
Colonnes du fichier :
nom_prenom — Nom et prénom (ex : MARTIN Pierre)
telephone — Numéro de téléphone
email — Adresse email (optionnel)
adresse — Adresse postale (optionnel)
mode_paiement — virement / cheque / especes (optionnel)
montant_paye — Total versé sur l'année (optionnel, ex : 120.00)
jan, fev, mars, avr, mai, juin, juil, aout, sept, oct, nov, dec — Montant par mois si disponible (optionnel)
🏫 Import familles & élèves ⬇ Template CSV
Colonnes du fichier :
nom_famille — Nom du contact principal (ex : MARTIN Pierre)
tel_papa, tel_maman — Téléphones (optionnels)
email — Email famille (optionnel)
enfant1_prenom, enfant1_niveau — 1er enfant (niveau : NP / N1 / N2 / N3 / N4 / N5)
enfant2_prenom, enfant2_niveau — 2e enfant (optionnel)
enfant3_prenom, enfant3_niveau — 3e enfant (optionnel)
montant_verse — Total versé sur l'année scolaire (optionnel)
sept, oct, nov, dec, jan, fev, mars, avr, mai, juin — Montant par mois si disponible (optionnel)
💰 Import historique trésorerie ⬇ Template CSV
⚠️ Usage unique — historique passé uniquement
Cet import sert à renseigner les transactions passées (avant la mise en service de l'application).
Pour les relevés bancaires mensuels, utilisez Trésorerie → 🏦 Import bancaire.
Colonnes du fichier :
date — Format JJ/MM/AAAA (ex : 15/09/2024)
libelle — Description de la transaction
montant_entree — Montant reçu (laisser vide si sortie)
montant_sortie — Montant dépensé (laisser vide si entrée)
categorie — don / cotisation / loyer / salaire / facture / ecole / projet / subvention / autre
mode_paiement — virement / cheque / especes / autre (optionnel)
note — Commentaire libre (optionnel)
